# Advanced Algorithms & Functional Programming in Java

This repository contains a collection of high-level algorithmic challenges focused on different programming paradigms, efficiency, and computational logic. 

The project demonstrates proficiency in translating complex mathematical problems into optimized Java code, comparing iterative, recursive, and functional approaches.

## Key Concepts Demonstrated

* **Functional Programming:** Extensive use of Java `Streams`, `UnaryOperator`, `Collectors`, and lambda expressions to handle data pipelines.
* **Recursion Strategies:** * **Tail Call Optimization (Recursive Final):** Designing memory-efficient recursive functions.
    * **Memoization:** Implementing recursion with memory to optimize time complexity in complex calculations.
* **Big Number Handling:** Using `BigInteger` and `Double` for high-precision scientific computing and large-scale factorials/sequences.
* **Performance Comparison:** Each exercise compares different computational methods (Iterative vs. Recursive vs. Functional) to analyze space and time complexity.

## Project Structure

* **Ejercicio 1 (Functional Streams):** Complex data grouping and transformation using iterative generation and functional mapping.
* **Ejercicio 2 (List Processing):** Implementation of sequence generation through multiple paradigms.
* **Ejercicio 3 (Memory Optimization):** Focus on **Dynamic Programming** principles (Recursive with vs. without memory/memoization).
* **Ejercicio 4 & 5 (Computational Precision):** Handling large numerical growth and performance benchmarking between iterative and recursive models.

## Technologies Used
* **Language:** Java 17+
* **Key Libraries:** `java.util.stream`, `java.math.BigInteger`, `java.util.function`.

## How to Use
Each class contains static methods representing different algorithmic solutions. To test the performance or output:
1. Clone the repository.
2. Compile the files using `javac ejercicios/*.java`.
3. Run the test suite (if available) or call the methods from a `Main` class.
